# Roguelike game

Randomly generated roguelike game

## Instructions
* Move - Arrows (up, down, left, right)
* Attack - Space

## Key features:
* Randomly generated dungeon.
* Point Light.
* Randomly spawning enemies.
* Enemies follow player.
* Infinite levels.
* Enemies can attack player.
